Тестовое задание FullStack разработчику
Необходимо реализовать сервис со следующим функционалом и применением следующих технологий.
Технологии:
Backend – Laravel
Frontend – ReactJS/VueJS
Database - PostgreSQL
Функционал:
В базе данных Lombard необходимо реализовать таблицу loanProducts со следующими колонками:
Id – Ключ
Name – Наименование объекта залогового имущества (например Iphone)
LoanDate – Дата начала залога
LoanDateEnd – Дата окончания залога
LoanSum – Сумма залогового имущества
LoanRealizationSum – Планируемая сумма реализации залогового имущества
PSK – Процентная годовая ставка
Должна быть консольная команда по добавлению данных в таблицу loanProducts (добавить не менее 10 записей)
Реализовать 2 API метода:
GET /Products – должен возвращать все записи из таблицы loanProducts.
Update /PSK – должен обновлять данные по процентной ставке в таблице loanProducts.

На клиентской части реализовать поле ввода с кнопкой обновить для годовой процентной ставки.
Ниже отрисовать все полученные данные в табличной форме.
При изменении годовой процентной ставки подсветить красным в таблице те поля где (LoanSum+((LoanDateEnd – LoanDate)*PSK))< LoanRealizationSum и подсветить зеленым (LoanSum+((LoanDateEnd – LoanDate)*PSK))> LoanRealizationSum
